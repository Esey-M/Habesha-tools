How to Run This Project

This is a Symfony application that requires proper server setup. To run it locally:

  1. Clone the repository
  2. Install PHP 8.1 or higher
  3. Install Composer dependencies
  4. Set up your database
  5. Configure your environment variables

For detailed instructions, please check the repository README.

When running locally, uploaded images are automatically deleted after 7 days to maintain system performance.
